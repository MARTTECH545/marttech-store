Using Installer

  1. Go to URL www.yourdomain.com/install and run the Installer
  2. Check the requirements and then add the database details
  3. Delete /public/install folder after installation

Manual Installation

  1. Copy all the files into your root directory . (You cant install it in sub directory , either install it on main domain or sub domain).
  2. Change database details from .env file , if you are viewing .env file from cpanel it will be hidden, select show hidden files while opening file manager.
  3. Import Booster.sql

Setting up Paypal

  1. Go to https://developer.paypal.com/ and log in .
  2. Create a app you will sandbox and live details .
  3. Set live CLIENT_ID and SECRET in .env file .

Setting up Stripe

  1. Go to https://stripe.com/ and log in .
  2. Copy the credentials provided by stripe .
  3. Set live STRIPE_API_SECRET and STRIPE_API_KEY in .env file .

Setting up Facebook App id

  1. Go to https://facebook.com/ and log in .
  2. Create a Facebook App .
  3. Set facebook APP id in \resources\views\social.blade.php line no 9

Setting up Mail

  1. Set mail settings from .env file
